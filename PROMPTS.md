# Prompt Guide: แบบสอบถามธุดงค์วัดป่าร้อยปี

คู่มือ Prompt สำหรับใช้งาน MCP Server วิเคราะห์แบบสอบถาม "หลังการสอบภาคสนามธุดงค์"
- สถานที่: วัดป่าร้อยปีหลวงพ่อวิริยังค์ จ.ราชบุรี
- วันที่: 12-15 ธันวาคม พ.ศ. 2568
- ผู้ตอบ: 804 คน

---

## สำหรับผู้บริหาร (Executive Summary)

### ภาพรวมและสถิติ
```
แสดงภาพรวมแบบสอบถามธุดงค์
```
```
สรุปความพึงพอใจทุกหมวดพร้อมคะแนนเฉลี่ย
```
```
คะแนนเฉลี่ยด้านความรู้ที่ได้รับทั้งหมด
```

### การวิเคราะห์เชิงกลยุทธ์
```
สรุปจุดแข็งและจุดอ่อนของการจัดงานธุดงค์ปีนี้
```
```
วิเคราะห์ความพึงพอใจทุกหมวดพร้อมข้อเสนอแนะเชิงนโยบาย
```
```
สรุป 5 ประเด็นสำคัญที่ต้องปรับปรุงสำหรับปีหน้า
```
```
จากข้อมูลทั้งหมด แนะนำ 3 สิ่งที่ควรทำเหมือนเดิม และ 3 สิ่งที่ควรปรับปรุง
```

### การเปรียบเทียบกลุ่ม
```
เปรียบเทียบความพึงพอใจด้านการจัดงานระหว่างนักศึกษา คณะทำงาน และผู้สังเกตการณ์
```
```
เปรียบเทียบคะแนนทุกด้านระหว่างนักศึกษากับคณะทำงาน แล้วสรุปความแตกต่าง
```
```
กลุ่มไหนให้คะแนนอาหารสูงสุด
```

### สิ่งที่โดดเด่น
```
อะไรคือสิ่งที่ผู้เข้าร่วมประทับใจมากที่สุด
```
```
มีปัญหาอะไรที่ถูกพูดถึงบ่อยที่สุด
```

### รายงานสรุป
```
สรุปแบบสอบถามเป็นรายงาน 1 หน้า สำหรับนำเสนอผู้บริหาร
```

---

## สำหรับคณะทำงาน (Operations & Improvement)

### ด้านสิ่งอำนวยความสะดวก

#### ห้องน้ำ
```
รวมข้อเสนอแนะเรื่องห้องน้ำทั้งหมด
```
```
มีคนพูดถึงห้องน้ำอะไรบ้าง
```

#### อาหาร
```
มีคนพูดถึงอาหารอะไรบ้าง ทั้งชมและติ
```
```
ค้นหาความเห็นเกี่ยวกับอาหาร
```
```
ข้อเสนอแนะเรื่องอาหารและเครื่องดื่ม
```

#### ที่พัก
```
ข้อเสนอแนะเกี่ยวกับที่พัก/เต้นท์มีอะไรบ้าง
```
```
หาข้อเสนอแนะที่เกี่ยวกับที่พัก
```

#### สถิติรวม
```
สถิติความพึงพอใจด้านสิ่งอำนวยความสะดวก แยกตามหัวข้อ
```
```
สรุปความพึงพอใจด้านสิ่งอำนวยความสะดวก
```

### ด้านการจัดงาน

#### กำหนดการ
```
รวมข้อเสนอแนะเรื่องกำหนดการและเวลา
```
```
รวมข้อเสนอแนะเรื่องกำหนดการ
```

#### เส้นทาง
```
ค้นหาความเห็นเกี่ยวกับเส้นทางเดินธุดงค์
```
```
ข้อเสนอแนะเกี่ยวกับเส้นทางมีอะไรบ้าง
```

#### ศาสนพิธี
```
ค้นหาความเห็นเกี่ยวกับศาสนพิธี
```

#### สถิติรวม
```
สถิติความพึงพอใจหมวดการจัดงาน เฉพาะนักศึกษา
```

### ด้านการประสานงาน
```
ข้อเสนอแนะเรื่องการประชาสัมพันธ์และประสานงาน
```
```
มีคนพูดถึงการประสานงานอย่างไรบ้าง
```

### ด้านพี่เลี้ยง
```
รวมสิ่งที่ประทับใจเกี่ยวกับพี่เลี้ยง
```
```
ข้อเสนอแนะสำหรับพี่เลี้ยงมีอะไรบ้าง
```
```
ค้นหาความเห็นเกี่ยวกับพี่เลี้ยง
```

### ปัญหาที่พบบ่อย
```
มีปัญหาอะไรที่ถูกพูดถึงบ่อยที่สุด
```
```
วิเคราะห์ข้อเสนอแนะทั้งหมด จัดกลุ่มเป็นหมวดหมู่ พร้อมจำนวนที่พูดถึง
```

## สำหรับผู้เข้าสอบ / นักศึกษา (Reflection & Sharing)

### สิ่งที่ประทับใจ

#### การเดินธุดงค์
```
แสดงสิ่งที่เพื่อนๆ ประทับใจเกี่ยวกับการเดินธุดงค์
```
```
แสดงสิ่งที่ประทับใจเกี่ยวกับการเดินธุดงค์ 10 อันดับแรก
```

#### พระอาจารย์
```
รวมสิ่งที่ประทับใจเกี่ยวกับพระอาจารย์
```
```
ค้นหาสิ่งที่ประทับใจเกี่ยวกับพระอาจารย์
```

#### กัลยาณมิตร
```
ค้นหาสิ่งที่ประทับใจเกี่ยวกับกัลยาณมิตร
```
```
มีใครพูดถึงเพื่อนร่วมธุดงค์บ้าง
```

#### ธรรมะ
```
มีใครพูดถึงธรรมะที่ได้รับบ้าง
```
```
ค้นหาสิ่งที่ประทับใจเกี่ยวกับธรรมะ
```

#### สถานที่
```
ค้นหาสิ่งที่ประทับใจเกี่ยวกับสถานที่
```
```
ข้อเสนอแนะเกี่ยวกับสถานที่มีอะไรบ้าง
```

### ความรู้ที่ได้รับ
```
สถิติความรู้ที่นักศึกษาได้รับจากการธุดงค์
```
```
คะแนนเฉลี่ยด้านความรู้ที่ได้รับทั้งหมด
```
```
เรื่องสมาธิ นักศึกษาให้คะแนนอย่างไร
```

### คุณธรรมจริยธรรม
```
คะแนนเฉลี่ยด้านคุณธรรมจริยธรรมทั้งหมด
```
```
สถิติด้านคุณธรรมที่นักศึกษาได้รับ
```

## Prompt วิเคราะห์เชิงลึก (Advanced Analysis)

### การเปรียบเทียบ
```
เปรียบเทียบความพึงพอใจด้านสิ่งอำนวยความสะดวก ระหว่างนักศึกษากับคณะทำงาน
```
```
เปรียบเทียบคะแนนการจัดงานระหว่างกลุ่มผู้ตอบ
```

### การจัดกลุ่มและวิเคราะห์
```
วิเคราะห์ข้อเสนอแนะทั้งหมด จัดกลุ่มเป็นหมวดหมู่ พร้อมจำนวนที่พูดถึง
```
```
สรุปประเด็นที่ต้องปรับปรุงสำหรับปีหน้า
```

### รายงานสรุป
```
สรุปแบบสอบถามเป็นรายงาน 1 หน้า สำหรับนำเสนอผู้บริหาร
```
```
สร้างรายงานสรุปผลการประเมินงานธุดงค์
```

## Keywords ที่ค้นหาได้

### สิ่งที่ประทับใจ (impressed)
| หมวด | คำค้นหา |
|------|---------|
| บุคคล | พี่เลี้ยง, พระอาจารย์, กัลยาณมิตร, คณะทำงาน, เพื่อน |
| กิจกรรม | เดินธุดงค์, สมาธิ, ธรรมะ, ศาสนพิธี |
| สถานที่ | สถานที่, บรรยากาศ, ธรรมชาติ, วัด |
| อื่นๆ | อาหาร, ที่พัก |

### ข้อเสนอแนะ (suggestion)
| หมวด | คำค้นหา |
|------|---------|
| สิ่งอำนวยความสะดวก | ห้องน้ำ, อาหาร, ที่พัก, เต้นท์, น้ำ |
| สถานที่ | หิน, พื้น, ทาง, เส้นทาง |
| การจัดงาน | กำหนดการ, เวลา, คิว, ประชาสัมพันธ์ |
| อื่นๆ | สุนัข, แสง, เสียง |

## ตัวอย่างการใช้งาน

### ค้นหาแบบเจาะจง
```
ค้นหาข้อเสนอแนะที่เกี่ยวกับ "คิว"
```
```
ค้นหาสิ่งที่ประทับใจเกี่ยวกับ "สมาธิ"
```

### ดูสถิติเฉพาะกลุ่ม
```
สถิติความพึงพอใจด้านการจัดงาน เฉพาะคณะทำงาน
```
```
สถิติด้านสิ่งอำนวยความสะดวก เฉพาะผู้สังเกตการณ์
```

### จำกัดจำนวนผลลัพธ์
```
แสดงสิ่งที่ประทับใจ 5 อันดับแรก
```
```
รวมข้อเสนอแนะเรื่องอาหาร 10 รายการ
```

---

## หมายเหตุ

- **Scale คะแนน**: 5 = มากที่สุด/พอใจมากที่สุด, 1 = น้อยที่สุด/พอใจน้อยที่สุด
- **กลุ่มผู้ตอบ**: นักศึกษา (690), คณะทำงาน (105), ผู้สังเกตการณ์ (9)
- **หมวดความรู้และคุณธรรม**: เฉพาะนักศึกษาเท่านั้น
- **หมวดการจัดงานและสิ่งอำนวยความสะดวก**: ทุกกลุ่ม
